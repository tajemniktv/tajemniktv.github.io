# Modding for UX and Performance

There is something satisfying about taking a game that is *almost* perfect and fixing that one UI element that drives you crazy.

## The Quest for Frames

Optimizing UI rendering can often lead to surprising performance gains. In my latest project, I found that simply reducing the complexity of the HUD layer improved 1% lows significantly.

## User Experience

It's not just about FPS. It's about flow. A cluttered UI breaks immersion. Modding allows us to strip away the noise and focus on the core experience.
